Method & honest caveats

tokens
"Tokens used" = the context the model must read to answer. Baseline = document + questions. ContextVault = compact ingest response + one task brief + one targeted expand per fact. Measured deterministically — isolated from agent reasoning overhead.
price
Input tokens only. Output is small and equal across methods, so it's excluded from the context-cost comparison.
<5k
Below ~5k tokens ContextVault costs more (0.2× at 1.5k) — exactly what the tool's own audit warns. The win is large documents, asked about repeatedly.
multi-turn
A real agent re-sends context each turn, so both methods cost more in practice — but the ratio holds, and ContextVault's reused brief makes follow-up questions far cheaper still.
